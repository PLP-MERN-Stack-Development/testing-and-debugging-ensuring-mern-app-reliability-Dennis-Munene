# mern-bug-tracker

## Overview
Small MERN-style bug tracker built for testing & debugging exercises. Contains:
- Backend (Express) with unit & integration tests (Jest + Supertest)
- Frontend (React) with component & integration tests (React Testing Library)
- Debugging exercises: console logs, Node inspector, Chrome DevTools, ErrorBoundary

## Quick start

### Backend
cd backend
npm install
npm run dev         # start server in dev mode
# or
npm start

## Run backend tests:
cd backend
npm run test

## Frontend
cd frontend
npm install
npm start

## Run frontend tests:
cd frontend
npm test
